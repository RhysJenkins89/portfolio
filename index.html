Skills

The core tools that I use are HTML, CSS, and JavaScript, but I’m also familiar with a variety of popular libraries and frameworks.

I always use the Sass preprocessor when I’m writing CSS, but if I want to use a dedicated CSS framework, I’ll jump over to Bootstrap or Tailwind. These tools help me keep my style sheets readable and easy to maintain.

When it comes to JavaScript, I’m most comfortable building robust, component-driven UIs with the React framework, but I also use the jQuery library to write cleaner, simpler JavaScript.
